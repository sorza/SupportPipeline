# SupportPipeline

Pipeline de múltiplos agentes construído com **Microsoft Agent Framework (MAF)** para oferecer suporte ao cliente de forma integrada e escalável.

## Visão Geral

O **SupportPipeline** organiza diferentes agentes especializados em um fluxo contínuo de atendimento.  
Cada agente é responsável por uma parte do processo (ex.: suporte técnico, análise de métricas, segurança), garantindo respostas mais rápidas, seguras e contextualizadas.

## Funcionalidades

- **Pipeline integrado**: conecta diversos agentes em sequência para resolver solicitações complexas.
- **Suporte contextualizado**: mantém histórico de conversas entre agentes e clientes.
